Installation

Download

We distribute a PHP Archive (PHAR) that contains everything you need in order to use PHPUnit. Simply download it from here, make it executable, and put it into your $PATH, for instance:

➜ wget http://raml2html.leanlabs.io/raml2html.phar
➜ chmod +x raml2html.phar
➜ sudo mv raml2html.phar /usr/local/bin/raml2html
➜ raml2html --version

You can also immediately use the PHAR after you have downloaded it, of course:

➜ wget http://raml2html.leanlabs.io/raml2html.phar
➜ php raml2html.phar --version

Usage

Run generator documentation:

➜ raml2html generate -i input_file.raml -o output_file.html

After generate open in your brouser for view output_file.html
